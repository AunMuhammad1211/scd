SETUP INSTRUCTIONS FOR NEW PC:
============================================

1. Copy Week7 folder to new PC

2. DELETE these folders/files BEFORE copying (to save space):
   - Week7/backend/node_modules/
   - Week7/backend/dist/
   - Week7/frontend/node_modules/
   - Week7/frontend/.next/
   - Any .env.local files

3. After copying to new PC, run:
   
   Backend:
   $ cd Week7/backend
   $ npm install
   $ npm run start:dev
   
   Frontend:
   $ cd Week7/frontend
   $ npm install
   $ npm run dev

4. Backend runs on: http://localhost:3000
5. Frontend runs on: http://localhost:3000 (Next.js dev server)
